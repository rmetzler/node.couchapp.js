# Installation

Install node.

Install npm.

<pre>
$ git clone https://github.com/mikeal/node.couchapp.js.git
$ cd node.couchapp.js
$ npm link .
</pre>

<pre>
$ couchapp help
-d,  --design :: File or directory for design document(s)
-t,  --test :: Run tests.
-s,  --sync :: Sync with CouchDB.
-c,  --couch :: Url to couchdb.
</pre>
